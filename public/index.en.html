Approach

We work in 3 dimensions

1. Continuous bioprocessing

All you need is a suspension growing cell. You don't need to invest in expensive continuous bioreactors to answer these questions. We have the knowledge to set a simplified-continuous process with all the elements that you already have in your lab.

2. Infrastructure

Our solution combines hardware and software bases to create modular production systems, that is, we deliver an adaptable and economical structure following the Asset Light principles. This structure connects to our own Operating System to produce fermentation in two stages.

3. Scalability

By having a modular infrastructure, we can satisfy the needs of the three actors regardless of the stage they are in, be it laboratory, pilot or commercial demonstration scale. By increasing productive capacity, we increase autonomy for the creation of new products in the FoodTech industry.

Feasibility Test

Before moving into continuous bioprocessing there are three main questions that need to be answered:

1. Do you have a suspension-growing cell?
2. Is the productivity of the system stable over time?
3. Is the system genetically stable over time?

Answering these questions can reduce the risks of failure of your project.

We have a tool that can answer these questions.
